## Szybki hosting (bez konfiguracji)
1) GitHub Pages: wrzuć pliki do repozytorium i włącz Pages dla gałęzi `main` / folderu root.
2) Netlify: przeciągnij folder na netlify.com (drag&drop).
3) Dowolny hosting: wrzuć pliki do katalogu public (www).
